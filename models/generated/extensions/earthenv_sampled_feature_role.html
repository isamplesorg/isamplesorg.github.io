Earth and Environmental Science extension - Sampled feature role

Terms to categorize the relation of a sampled feature to its context. In the Earth Science realm this is typically relation of sampled feature to a containing rock body or rock body part.

Published

January 7, 2024

Vocabularies and extensions:

1 Sampled feature part

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sampledfeaturepart
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

This is a top concept of the vocabulary.

Immediately narrower concepts: inclusion, individual constituent, rock body

Definition: A sampled feature that is part of a larger sampled feature; Could be improper part (e.g. the whole larger feature)

1.1 inclusion

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/inclusion
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion

Immediately narrower concepts: clast, cognate inclusion, fluid inclusion, melt inclusion, mineral inclusion, xenolith

Definition: sampled feature is an object contained within a larger body of material.

Example: fluid inclusion, xenolith, clast in a sedimentary material.

1.1.1 clast

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/clast
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion->clast

Definition: A clast is a fragment of rock or mineral derived from some pre-existing rock or generated by intrabasinal processes that is a part of a sediment

1.1.2 cognate inclusion

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/cognateinclusion
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion->cognate inclusion

Definition: An inclusion in an igneous rock that is interpreted to be genetically related to the enclosing rock.

Alternate labels: autolith

1.1.3 fluid inclusion

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/fluidinclusion
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion->fluid inclusion

Definition: an inclusion that is an opening in a mineral grain containing fluid (liquid or gas).

1.1.4 melt inclusion

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/meltinclusion
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion->melt inclusion

Definition: An inclusion in a mineral grain in an igneous rock that is interpreted to consist of trapped magma from which the igneous rock was formed.

1.1.5 mineral inclusion

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/mineralinclusion
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion->mineral inclusion

Definition: An inclusion of a mineral phase contained inside a grain of another mineral.

1.1.6 xenolith

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/xenolith
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->inclusion->xenolith

Definition: An inclusion that is a fragment of pre-existing material that is contained in an igneous rock and unrelated to the magma that formed the rock.

1.2 individual constituent

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/individualconstituent
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->individual constituent

Immediately narrower concepts: cement, groundmass, phenocryst, porphyroblast

Definition: sampled feature is a single particle or collection of particles that share some characteristic, within or extracted from a larger sample.

Example: mineral separate, point analysis of particular particles

1.2.1 cement

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/cement
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->individual constituent->cement

Definition: sampled feature is material that binds constituent particles together in a sedimentary rock.

1.2.2 groundmass

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/groundmass
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->individual constituent->groundmass

Definition: sampled feature is fine-grained to aphanitic material that is interstitial to larger crystals or particles in a sample.

1.2.3 phenocryst

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/phenocryst
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->individual constituent->phenocryst

Definition: Sampled feature is a crystal that is significantly larger than surrounding crystals in an igneous rock.

1.2.4 porphyroblast

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/porphyroblast
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->individual constituent->porphyroblast

Definition: Sampled feature is a crystal that is significantly larger than surrounding crystals in a metamorphic rock.

1.3 rock body

URI https://w3id.org/isample/earthenv/essampledfeatrole/1.0/rockbody
defined in vocabulary https://w3id.org/isample/earthenv/essampledfeatrole/1.0/sfrolevocabulary

Path from the top concept: Sampled feature part->rock body

Definition: sampled feature is representative of a rock body, i.e. it represents the whole of the sampled body.
