Sampled Feature Type vocabulary

Categories to specify the broad context that a sample is intended to represent.

Published

December 5, 2025

Vocabularies and extensions:

  • Sampled Feature Type vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

History:

2021-07-09 Remove Marine biome, Subaerial terrestrial environment, Subaqueous terrestrial environment per github issue https://github.com/isamplesorg/metadata/issues/41. Make Experiment setting and Laboratory or curatorial environment subclasses of Active human occupation site.
2021-12-10 SMR add missing skos:inScheme statements
2022-01-07 SMR change to https://w3id.org/isample/ uri base, make the ConceptScheme an ontology as well. For uploading to ESIP COR and w3id resolution redirect set up. Add some mappings to other ontologies using seeAlso, closeMatch, narrowMatch.
2022-03-11 SMR change definitions from rdfs:comment to skos:definition. Minor fixes in definitions. Add skos matches to URIs from other vocabularies. Fix typo in glacierenvrionment URI (changed the URI to glacierenvironment)
2022-09-07 SMR update some of the skos mappings to other vocabularies; remove references to other vocabularies as NamedIndividual. Remove rocksample class, it was not linked in hierarchy and inconsistent with design.
2022-09-30 add biological entity as sampled feature, per issue https://github.com/isamplesorg/metadata/issues/107. This update was lost at some point and added back in 2022-12-09.
2023-11-05 SMR update version to 1.0, prep for release
2024-04-19 SMR update definitions to remove use of ‘specimen’. Edit some definitions for better clarity
2024-09-13 remove version numbers from URI

1 Any sampled feature

URI https://w3id.org/isample/vocabulary/sampledfeature/anysampledfeature
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

This is a top concept of the vocabulary.

Immediately narrower concepts: Anthropogenic environment, Biological entity, Earth environment, Extraterrestrial environment

Definition: Any thing that can be sampled. Top concept in sampled feature type vocabulary.

1.1 Anthropogenic environment

URI https://w3id.org/isample/vocabulary/sampledfeature/anthropogenicenvironment
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Anthropogenic environment

Immediately narrower concepts: Active human occupation site, Site of past human activities

Definition: Sampled feature is produced by or related to human activity past or present.

1.1.1 Active human occupation site

URI https://w3id.org/isample/vocabulary/sampledfeature/activehumanoccupationsite
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Anthropogenic environment->Active human occupation site

Immediately narrower concepts: Experiment setting, Laboratory or curatorial environment

Definition: sampled feature is a site at which there are ongoing human activities

1.1.1.1 Experiment setting

URI https://w3id.org/isample/vocabulary/sampledfeature/experimentsetting
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Anthropogenic environment->Active human occupation site->Experiment setting

Definition: Sampled feature is an experimental set up that produced the sample; the sample is the product of the experiment.

1.1.1.2 Laboratory or curatorial environment

URI https://w3id.org/isample/vocabulary/sampledfeature/laboratorycuratorialenvironment
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Anthropogenic environment->Active human occupation site->Laboratory or curatorial environment

Definition: Sampled feature is a laboratory or other research site, collected with intention of characterizing the environment in which data are collected or other research conducted, that might affect results or safety; e.g. lab blank measurements.

1.1.2 Site of past human activities

URI https://w3id.org/isample/vocabulary/sampledfeature/pasthumanoccupationsite
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Anthropogenic environment->Site of past human activities

Definition: sampled feature is a place where humans have been and left evidence of their activity. Includes prehistoric and paleo hominid sites

1.2 Biological entity

URI https://w3id.org/isample/vocabulary/sampledfeature/biologicalentity
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Biological entity

Definition: Sampled feature is an organism. Use for samples that represent some species of organism as the proximate sampled feature, not the environment that the organism inhabits.

1.3 Earth environment

URI https://w3id.org/isample/vocabulary/sampledfeature/earthenvironment
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment

Immediately narrower concepts: Atmosphere, Earth interior, Earth surface, Glacier environment, Subsurface fluid reservoir, Water body

Definition: Sampled feature is the natural Earth environment

1.3.1 Atmosphere

URI https://w3id.org/isample/vocabulary/sampledfeature/atmosphere
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Atmosphere

Definition: Sampled feature is the Earth’s atmosphere

1.3.2 Earth interior

URI https://w3id.org/isample/vocabulary/sampledfeature/earthinterior
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Earth interior

Definition: Sampled feature is solid material from within the Earth

1.3.3 Earth surface

URI https://w3id.org/isample/vocabulary/sampledfeature/earthsurface
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Earth surface

Immediately narrower concepts: Lake river or stream bottom, Marine water body bottom, Subaerial surface environment

Definition: Sampled feature is the interface between solid earth and hydrosphere or atmosphere. Includes samples representing things collected on the surface, in the uppermost part of the material below the surface, or air or water directly at the contact with the Earth surface.

1.3.3.1 Lake river or stream bottom

URI https://w3id.org/isample/vocabulary/sampledfeature/lakeriverstreambottom
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Earth surface->Lake river or stream bottom

Definition: Sampled feature is the interface between the solid Earth interface and a lake or flowing water body.

1.3.3.2 Marine water body bottom

URI https://w3id.org/isample/vocabulary/sampledfeature/marinewaterbodybottom
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Earth surface->Marine water body bottom

Definition: Sampled feature is the interface between the solid Earth and a marine or brackish water body. Includes benthic boundary layer: the bottom layer of water and the uppermost layer of sediment directly influenced by the overlying water.

Alternate labels: Sea floor

1.3.3.3 Subaerial surface environment

URI https://w3id.org/isample/vocabulary/sampledfeature/subaerialsurfaceenvironment
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Earth surface->Subaerial surface environment

Definition: sampled feature is the interface between solid Earth and atmosphere. Sample is collected on the surface, or immediately below surface (zone of bioturbation). Include soil and recently deposited subaerial sediment at the surface.

1.3.4 Glacier environment

URI https://w3id.org/isample/vocabulary/sampledfeature/glacierenvironment
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Glacier environment

Definition: Sampled feature is a glacier, ice sheet, ice shelf, iceberg, or rock or water directly under or on top of such ice.

1.3.5 Subsurface fluid reservoir

URI https://w3id.org/isample/vocabulary/sampledfeature/subsurfacefluidreservoir
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Subsurface fluid reservoir

Definition: Sampled feature is fluid that resides in fractures, intergranular porosity or other open space in the solid earth.

1.3.6 Water body

URI https://w3id.org/isample/vocabulary/sampledfeature/waterbody
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Water body

Immediately narrower concepts: Marine environment, Terrestrial water body

Definition: Sampled feature is the Earth’s hydrosphere.

1.3.6.1 Marine environment

URI https://w3id.org/isample/vocabulary/sampledfeature/marinewaterbody
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Water body->Marine environment

Definition: Sampled feature is the marine hydrosphere.

1.3.6.2 Terrestrial water body

URI https://w3id.org/isample/vocabulary/sampledfeature/terrestrialwaterbody
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Earth environment->Water body->Terrestrial water body

Definition: Sampled feature is terrestrial hydrosphere– lake, other standing water, or a flowing water body (river, stream..). Include saline water in terrestrial evaporite environments.

1.4 Extraterrestrial environment

URI https://w3id.org/isample/vocabulary/sampledfeature/extraterrestrialenvironment
defined in vocabulary https://w3id.org/isample/vocabulary/sampledfeature/sampledfeaturevocabulary

Path from the top concept: Any sampled feature->Extraterrestrial environment

Definition: Sampled feature is the environment outside of solid earth, hydrosphere, or atmosphere.
